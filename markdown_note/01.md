## JSX语法

jsx是js的语法扩展（eXtension)

jsx的书写规范：

- jsx结构中只能有一个根元素
- jsx会包含一个（）,将整个jsx当作一个整体，方便阅读
- 单标签必须以/>结尾

注释：

```
{/*注释*/}
```

jsx变量作为子元素插入：

object不能作为react的子元素



jsx绑定属性

绑定class属性：避免和js中的class类关键字误会 叫做className

- 字符串拼接
- 数组

{{color:"red"}} 里面的括号代表的是对象 属性值必须要用引号括起来，否则就是变量
